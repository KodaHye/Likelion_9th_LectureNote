**MTV패턴 이해하기**

* MTV는 위의 세 가지 문제를 방지하기 위해서도 사용됨

* MTV패턴: 장고의 디자인 패턴 (Model, Template, View)

  * Tempate: Front End
    * 사용자가 보이는 영역
    * HTML p
    * CSS
    * 템플릿
    * 좀 더 나아가면 JS로 동작까지 컨트롤 할 수 있음
  * Model, View: Back End
    * Model: DataBase(DB)
    * View: 데이터를 처리하는 곳, MTV 중에서 핵심
