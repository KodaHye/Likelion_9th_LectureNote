##### # HTML(1)

* HTML(**H**yper **T**ext **M**arkup **L**anguage)
  * Hyper Text = Link
  * Markup Language: 컴퓨터가 이해할 수 있는 언어
* HTML 문서를 웹 서버로 전한다고 할 때!
  * 문서를 보이는 그대로 전달 → 서버는 문서 그대로 받아들이지 못함
  * 문서를 보낼 떄 하나하나 글이 어떤 글인지, 어떤 언어로 작성되었는지 전부 다 설명해야됨
    1. 글                                                  
    2. 글을 감싸는 틀 <제목>, <한 문단>     >     태그
    3. 틀에 붙는 부가설명                           >     속성
